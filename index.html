RECENT PROJECTS

Safe Space

A tool to document domestic violence and remain safe

Safe Space

  • JavaScript
  • Node.JS
  • Mobile

Safe Space is working mock-up for Empath360, a non-profit domestic violence center. Safe Space is a mobile app that allows users to store information about domestic violence experiences for future court dates. For safety the app is hidden behind a news feed with "quick button return" to hide the app. It also allows users to call 911 or a safety contact with one button if they are in danger. It is currently deployed on Heroku.

Quiz Time

A quiz game that will track and average your scores

Quiz Time

  • Rudy on Rails
  • HTML/CSS
  • Bootstrap

Quiz-Time is a project that allows users to sign up and take quizes, track their scores and battle for the top score. Admins are also able to write and publish new quizes. It is currently deployed on Heroku.

Pando-Ra

Pando-Ra

  • HTML/CSS
  • JavaScript
  • RESTful API

Pando-Ra is a project that searches the SoundCLoud API and displays artist and tracks and give the user the ability to play the song directly on the page.
